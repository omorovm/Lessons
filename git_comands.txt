локальный репозиторий - папка .git папка на ноуте
удаленный репозиторий - папка на сервере (которую мы создали на сайте)

git init - создает git репозиторий из текущей папки

git add -  добавляет папки и файлы для отслеживания (эту команду пишем каждый раз перед командой commit)

git commit -m "сообщение" - создает новую версию (сохраняет изменения), если были изменения

git status - показывает состояние файлов

git branch - показывает все ветки

git remote add origin {ssh|https} - создает связь между локальным и удаленным репозиторием

git push origin master - добавление файлов на удаленный репозиторий 

git checkout
